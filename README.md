## 上传代码命令


### 创建一个新的仓库
echo "# 6.828os_worrk" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:niuppp-chy/6.828os_worrk.git
git push -u origin main

### 上传文件到一个已存在的仓库

git remote add origin git@github.com:niuppp-chy/6.828os_worrk.git
